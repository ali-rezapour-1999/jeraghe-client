## ⚡ نحوه اجرای پروژه

### **1. نصب پیش‌نیازها**

ابتدا اطمینان حاصل کنید که `bun` , `Node.js` روی سیستم شما نصب شده است.

```bash
# نصب وابستگی‌ها با npm
bun install

# بررسی استانداردهای کد
bun run lint

# فرمت‌دهی کدها
bun run format

```
